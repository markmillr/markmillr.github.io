Windows instructions

Mac instructions

I promise that getting the required software set up on your computer is by far the most complicated and difficult part of this class! After that, it's all drawing, robots, and games with working Python code.

I think it is best if parents do this setup, with or without their student (depending on personality). I realize that many of you may not have much experience with computers. Don't feel intimidated, and don't hesitate to ask for help if something's not working. These instructions have been tested with Windows 10 and macOS Mojave and should work for anybody who has those systems. Other, older operating systems may encounter issues... the best approach is just to try the instructions and hope that it works. If you have an older computer, you may need to consider purchasing a more recent laptop for things to work properly. For more information on this see Mark's opinionated guide to buying a laptop (Feb 2019). However before you go that route, please feel free to call or text me for help if you have any difficulties!

In a nutshell, we need to 1) install the Sublime text editor, and 2) install Anaconda (fairly large, takes ~30 minutes to download). Then we need to 3) check to make sure that the Turtle library is installed so that we can draw cool stuff with Python.

Instructions for both Windows and macOS are provided here.

About the software we're installing:

Sublime Text is a programming-specific text editor for writing code, and Anaconda is a package manager that started in data science but is now used by professionals in many fields to make it easy to do real work with Python. Anaconda will install Python 3.7 for us as well as a suite of useful libraries (Python tools/resources) such as NumPy and Matplotlib. Turtle is a Python drawing package that is useful for helping people to learn to code in Python.
