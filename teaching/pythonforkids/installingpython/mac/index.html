Installation of required software before class (macOS):

First determine what version of Mac operating system you have (Mojave, High Sierra, etc.), and write this information down, because you may need it. Find out which macOS your Mac is using. Generally, it's best to have the latest possible version, which is currently Mojave (macOS 10.14.3).

We'll also need to find out whether our Mac is the 32-bit or 64-bit version, and write this down, because we may need it. 32-bit Macs are just about obsolete now and will probably not be able to correctly run the required software. How to Tell if Your Mac is 32-bit or 64-bit.

Go to Sublime Text's website and download Sublime Text 3 for Mac.



Click on "DOWNLOAD FOR MAC", then double click on the Sublime Text dmg package that appears in your Downloads. In Safari this looks like:



In the window that appears, drag the "Sublime Text.app" to the "Applications" folder:

2) Install Anaconda

Go to Anaconda's download website and download Anaconda for Mac.



Download the Python 3.7 version for Mac and Anaconda will start downloading. This file is over 650 MB and could take a half hour or more, depending on your internet speed.

Double click on the Anaconda package file in the Downloads area (in the upper right corner of Safari).



Anaconda is considered to be a trustworthy source so say yes to trusting their software...



Click through the license and questions to proceed with the Anaconda Installation.



This may take 10-20 minutes and will require a lot of storage, more than 2 GB.



You don't need Microsoft VSCode, so skip through this dialogue and close the Anaconda installer.



Now we're done installing Anaconda:



Python 3.7 should be installed on our machine now, so let's check by opening up the Terminal. The Terminal is how we talk to a Mac/Linux/Unix computer using the keyboard.

There are a couple of ways to open up Mac's Terminal application. One is to use the Launchpad and look or search for the Terminal icon:




Click through to the "Other" area of the Launchpad view to find the Terminal app:



Now a window containing the Terminal should appear. It will look something like this, possibly with different colors (these are customizable):

If Anaconda did what we expect it to do, then it should have installed Python 3.7 on our Windows machine and set our machine up so that we can start the Python interpreter (dialogue) by typing "python" at the Terminal prompt (the dollar sign "$"):



If you see something like the above screenshot, with Python 3.7 or above, SUCCESS! You have Python on your machine. If not, you will receive an error message. Get in touch with me if this hasn't worked.

Where you see ">>>" is the Python interpreter, a place where you can talk directly to your computer in Python. It's kind of like going to Mexico and knowing that it is a place where people speak Spanish.

The traditional first program in any programming language is to print "Hello, World!" to the output. Let's make this a special one by writing "Hello, Susanville!"

Type the following: print("Hello, Susanville!")

Then press enter.



Well done, you have successfully installed Python on your machine and run your first line of Python code! Most teachers agree that this part should be easier than it is, but the good news is that this is kind of the hardest part about getting started with Python.

Now check to see that Anaconda installed the Turtle drawing library for us by typing "import turtle" in the Python interpreter:



If you hit enter and get no error message and just another Python interpreter prompt (>>>), SUCCESS! Turtles has been installed, and we can soon draw our first patterns with Python.
