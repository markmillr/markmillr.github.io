Python For Kids - Susanville 2019

Installation of required software before class: In a nutshell, you need to 1) install the Sublime text editor, and 2) install Anaconda (fairly large, takes ~30 minutes to download), and then if not done automatically already, install Python 3.7 using Anaconda. Then you need to 3) install the Turtle library for learning Python by drawing cool stuff with code. Instructions for both Windows and macOS are provided here.

Sublime Text is a programming-specific text editor, and Anaconda is a platform that started in data science but is now used by professionals in many fields to make it easy to do real work with Python. Anaconda will install Python 3.7 and a suite of useful libraries (Python tools/resources) for us, such as NumPy and Matplotlib.

Windows instructions:

1) Install Sublime Text 3

Go to Sublime Text's website and download Sublime Text 3 for Windows.



Click on "DOWNLOAD FOR WINDOWS" and then click "Run" in the window that pops up at the bottom of the screen. Click "Yes" when asked if you would like to allow the installer to make changes to your computer. This will open up a Setup Wizard for Sublime Text 3:



Find Sublime Text 3 on your Windows computer and open it. This is where we will write our Python code with lots of pretty colors that are helpful for programming. I used Sublime Text to write the static HTML for this barebones, poorly written, bare minimum static webpage.



What you will get by downloading Sublime is a free trial that basically lasts forever. If you are feeling generous you can pay the developer who took the time to make Sublime Text by buying a license.

If you want to play with Sublime a little bit, write a haiku and save it as a ".txt" file.



2) Install Anaconda

Go to Anaconda's download website and download Anaconda for Windows.



Click "Run" and Anaconda will start downloading. This could take a half hour or more, depending on your internet speed.

Click through the buttons that appear in order to install Anaconda:



Make sure that BOTH "Add Anaconda to my PATH environment variable" AND "Register Anaconda as my default Python 3.7" are CHECKED. If not, things could get complicated.



Now it will take a few minutes to install Anaconda.



Find Anaconda (64-bit) in your Windows Menu and open the Anaconda Navigator:
