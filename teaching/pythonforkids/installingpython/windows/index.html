Python For Kids - Susanville 2019

Installation of required software before class (Windows):

First determine what version of Windows you have (XP, 8, 10, etc.), and write this information down, because you may need it. How to find out which Windows operating system you are running

We'll also need to find out whether our Windows is the 32-bit or 64-bit version, and write this down, because we may need it. How to determine whether a computer is running a 32-bit version or 64-bit version of the Windows operating system

1) Install Sublime Text 3

Go to Sublime Text's website and download Sublime Text 3 for Windows.



Click on "DOWNLOAD FOR WINDOWS", double check the filename to make sure that they are giving you the correct version (the 32-bit or 64-bit Windows version), and then click "Run" in the window that pops up at the bottom of the screen. Click "Yes" when asked if you would like to allow the installer to make changes to your computer. This will open up a Setup Wizard for Sublime Text 3:



Find Sublime Text 3 on your Windows computer and open it. Success! This is where we will write our Python code with lots of pretty colors that are helpful for programming. I used Sublime Text to write the static HTML for this barebones, poorly written, bare minimum static webpage.

What you will get by downloading Sublime Text is a free trial that basically lasts forever. If you are feeling generous you can pay the developer who took the time to make Sublime Text by buying a license to register it.

If you want to play with Sublime Text a little bit, write a haiku and save it as a ".txt" file.



In class we will use Sublime Text to write and edit Python files (".py").

2) Install Anaconda

Go to Anaconda's download website and download Anaconda for Windows. Make sure to download the 64-bit or 32-bit version, as appropriate.



Click "Run" and Anaconda will start downloading. This could take a half hour or more, depending on your internet speed.

Click through the buttons that appear in order to install Anaconda:



Make sure that BOTH "Add Anaconda to my PATH environment variable" AND "Register Anaconda as my default Python 3.7" are CHECKED. Ignore any warnings. If you don't check both boxes, things could get complicated.



Now it will take a few minutes to install Anaconda.



Find Anaconda in your Windows Menu and open the Anaconda Navigator:



Decide whether you want to share usage information with Anaconda. I don't think it matters much either way.



You can uncheck both boxes shown below:



You also don't need Microsoft VSCode, so skip through this dialogue.



Now this step will seem strange, but when Anaconda Navigator opens up, just close it because we don't really need it right now.

Python should be installed on our machine, so let's check it by opening up the Command Prompt. The Command Prompt is how we talk to a Windows computer using the keyboard.

Type "cmd" in the Windows search box and then click on the Command Prompt app that appears.



Now a window containing the Command Prompt should appear. It will look something like this:

If Anaconda did what we expect it to do, then it should have installed Python 3.7 on our Windows machine and set our machine up so that we can start the Python interpreter (dialogue) by typing "python" at the Command Prompt:



If you see something like the above screenshot, SUCCESS! You have Python on your machine. If not, you will receive an error message. Get in touch with me if this hasn't worked.

Where you see ">>>" is the Python interpreter, a place where you can talk directly to your computer in Python. It's kind of like going to Mexico and knowing that it is a place where people speak Spanish.

The traditional first program in any programming language is to print "Hello, World!" to the output. Let's make this a special one by writing "Hello, Susanville!" Type the following: print("Hello, Susanville!")
Then press enter.



Well done, you have successfully installed Python on your machine and run your first line of Python code! Most teachers agree that this part should be easier than it is, but the good news is that this is kind of the hardest part about getting started with Python.

Now check to see that Anaconda installed the Turtle drawing library for us by typing "import turtle" in the Python interpreter:



If you hit enter and get no error message and just another Python interpreter prompt (>>>), SUCCESS! Turtles has been installed, and we can soon draw our first patterns with Python.

We are technically done and ready for class, so can stop here if you like!
