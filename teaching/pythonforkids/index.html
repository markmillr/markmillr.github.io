Instructions on installing Python (Do this at home before coming to first class!)

Course content:

Students will learn to do the following, among other skills:

Describe how programming is required for specific types of businesses and industries.

Modify Python 3 code to change an image drawn with the Turtle module. Initially, the IDLE environment or Trinket.io will be used as a bridge to the command line.

Write and run simple Python commands for arithmetic and programs like Hello World!, What's Your Name? and Mad Libs. Initially, the IDLE environment or Trinket.io will be used as a bridge to the command line.

Navigate their computer's file structure and create, move, copy, and modify files using the command line.

Write Python scripts using the Sublime text editor.

Run Python scripts using the command line.

Describe how people lived and worked before and after the adoption of new computing technologies.

Write and run simple Turtle scripts using sequences of commands and loops to draw pretty pictures.

Write and run more complex Turtle scripts using sequences of commands and loops to draw pretty pictures.

Post their programmed art to a class webpage using Padlet.

Improve their typing speed.
